## 📝 注意事项

### 商务礼仪

1. 着装：正装（西装、领带）
2. 守时：提前10分钟到达
3. 礼貌：主动握手、递名片（双手）
4. 倾听：认真听客户需求，做好记录
5. 尊重：称呼职务，不打断客户讲话

### 沟通技巧

1. 开场：简短自我介绍，建立信任
2. 提问：多问开放式问题，了解真实需求
3. 展示：突出差异化优势，避免泛泛而谈
4. 互动：鼓励客户提问，及时回应
5. 收尾：总结关键点，确认下步行动

### 风险防范

1. 技术问题：提前准备FAQ，不确定的不要承诺
2. 竞争对手：不要诋毁对手，突出自身优势
3. 价格问题：先谈价值，再谈价格
4. 承诺问题：不要过度承诺，留有余地
5. 信息安全：不要泄露其他客户信息

### 应急预案

1. 客户临时取消：立即改约，不要浪费行程
2. 交通延误：提前通知客户，诚恳道歉
3. 设备故障：准备U盘备份，使用客户设备
4. 技术卡壳：诚实表示回去确认，不要瞎说
5. 突发问题：保持冷静，及时向公司请示
